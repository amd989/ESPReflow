What is ESP Forge?

ESP Forge is a wireless forge controller capable controlling a forge or a burnout oven.

Credits

  • It is built using Wemos D1 Mini module with Arduino framework at it's core.
  • WebUI is powered by Angular using Material.
  • Firmware is a fork from ESPReflow on GitHub
  • Favicon downloaded from www.iconfinder.com/icons
